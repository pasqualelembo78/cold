Impostiamo la variabile in apache2 

sudo nano /etc/apache2/envvars

Infondo al file 

export API_KEY="Desy2011@" 

sudo chown www-data:www-data /opt/mevacoin_config/.env
sudo chmod 600 /opt/mevacoin_config/.env

sudo systemctl reload apache2
# oppure
apt update && apt install php8.1-fpm
sudo systemctl reload php8.1-fpm

Ogni chiamata deve essere ora

 $apiKey = $API_KEY;

Mentre in wallet-api

Rendi permanente variabile d'ambiente 

echo "export \$(grep -v '^#' /opt/mevacoin_config/.env | xargs)" >> ~/.bashrc

E poi avvia con screen wallet-api 

screen -DmS wallet-api /opt/mevacoin/build/src/wallet-api --port 8070 --rpc-bind-ip 0.0.0.0 --enable-cors "*" --rpc-password "$API_KEY"
